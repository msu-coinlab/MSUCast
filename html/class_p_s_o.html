◆ optimize()

void PSO::optimize ( )

Optimizes using the PSO algorithm.

Runs the PSO optimization loop.

After initialization, the swarm is iterated for a maximum number of iterations. In each iteration, particles update their positions based on a randomly selected global best, the swarm is re-evaluated, and personal and global bests are updated.
