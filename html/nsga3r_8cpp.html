Detailed Description

Main implementation of the NSGA-III algorithm.

This file implements the NSGA-III routine including population initialization, evaluation, selection, crossover/mutation, merging, ranking and reporting. It also contains several scenario‐related functions used to load and process external configuration and scenario data.
