Detailed Description

Implementation of the EPA_NLP class and related functions for environmental optimization.

This file implements a nonlinear programming (NLP) formulation that uses scenario data (in JSON format) to optimize BMP (Best Management Practices) selection and application. It also uses Redis for ETA lookups and reads/writes Parquet/JSON/CSV files.
