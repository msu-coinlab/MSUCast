static CommandResult my_execute::Command::exec ( const std::string & command)
inlinestatic

Executes a system command and returns its STDOUT and exit status.

This function uses popen() to capture the command's STDOUT. If the command fails, a std::runtime_error is thrown.

Parameters
commandThe system command to execute.
Returns
CommandResult containing the command's STDOUT output and exit status.

Parameters
