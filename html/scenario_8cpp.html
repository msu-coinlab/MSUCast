Detailed Description

Implements the Scenario class and associated functions for scenario data handling, normalization, cost computation, and file I/O in the optimization process.

This file reads scenario and BMP data from JSON and CSV sources, computes efficiency, land conversion, animal and manure costs, and writes outputs in JSON and Parquet formats.

Environment variables (e.g., REDIS_HOST, MSU_CBPO_PATH) are used for configuration.

Computes total loads based on a list of parcel keys.

For each key in parcel_keys, uses the phi_dict and amount map to compute and sum load contributions for various pollutants.
