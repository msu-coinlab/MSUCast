Detailed Description

Represents a single CSV row containing multiple fields.

This class is effectively a "view" over the parser's raw data structures. Each field can be accessed via operator[] by index or by column name (if known).

Example

for (auto& row : reader) {
std::string col_0 = row[0].get<std::string>();
std::string col_name_val = row["Name"].get<std::string>();
}
